# run-kotlin-native

![Build binaries](https://github.com/karakani/run-kotlin-native/workflows/Build%20binaries/badge.svg)

## How to build

```shell script
./gradlew clean build
```

### Compile binaries for specific platform

for MacOS,

```shell script
./gradlew clean macosX64Binaries
```

for Linux (amd64),

```shell script
./gradlew clean linuxX64Binaries
```
